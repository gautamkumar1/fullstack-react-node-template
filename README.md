
## Frontend

**Tech Stack:**  
- React  
- TanStack Query  
- Better Auth  
- Shadcn UI  

**Environment Variables (`.env.example`):**
```bash
VITE_FRONTEND_DEV_URL=http://localhost:5173
VITE_BACKEND_DEV_URL=http://localhost:3000
```
## Backend

**Tech Stack:**  
- Nodejs  
- PostgreSQL (DB)  
- Prisma (ORM)  
- Better Auth (Authentication)

**Environment Variables (`.env.example`):**
```bash
PORT=3000
DATABASE_URL="" # PostgreSQL connection string (e.g., Neon DB)
BETTER_AUTH_SECRET= # Obtain this from the Better Auth dashboard
BETTER_AUTH_URL=http://localhost:3000 # Base URL of the Node.js backend
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```
